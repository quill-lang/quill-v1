Expand description

Represents the contents of an expression (which may or may not have been already type checked). The type V represents the type variables that we are substituting into this symbol. The type P represents a pattern. The type I represents the contents of an impl expression. You should use ExpressionContents or ExpressionContentsT instead of this enum directly.
