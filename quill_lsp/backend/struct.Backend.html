pub(crate) async fn get_project_root(
    &self,
    url: Url
) -> Option<(SourceFileIdentifier, ProjectRoot)>

Check whether the given URI is inside a Quill project. This function returns the project root folder, which can be used as a key in RootFileSystems, along with the identifier of the source file relative to the project root.

In particular, this checks parent URIs until a quill.toml file is found. If one is found, then that directory is added to RootFileSystems as a file system root.
