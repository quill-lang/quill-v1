pub struct DiagnosticResult<T> { /* fields omitted */ }
Expand description

Even if warnings or errors are emitted, we may still be able to continue parsing the code. So we need some kind of result type that allows us to raise errors or other messages while still retaining an ‘Ok’ state, as far as the rest of the code is aware.

Upon exiting the program, all error messages will be scanned to check the most severe error level. If any errors exist, no warnings will be emitted.
