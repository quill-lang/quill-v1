A basic block is a block of code that can be executed, and may manipulate values. Control flow is entirely linear inside a basic block. After this basic block, we may branch to one of several places.

ControlFlowGraph

After validation, the control flow graph must be in a topologically sorted order: we jump only from lower-indexed basic blocks to higher-indexed basic blocks. This means that, to trace control flow, ensuring that each variable is initialised before used, you can just iterate in order over the list of basic blocks.
